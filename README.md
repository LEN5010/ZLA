# 知行实验室答题助手 (Zhixing Laboratory Assistant)

## 简介

本脚本专为知行实验室 (mhlabs.cloudrange.cn) 平台设计，旨在辅助用户高效完成在线培训任务。脚本集成了自动化处理模块，能够识别并处理 Articulate 幻灯片课件与 Vue 架构的考试系统，大幅提升学习效率。

## 主要功能

### 1. 幻灯片自动化处理

- **深度扫描**：自动遍历页面内存，识别 Articulate Storyline 数据结构。
- **自动完成**：通过模拟请求，自动发送页面浏览记录，无需手动点击翻页。
- **跨域支持**：内置 iframe 通信机制，能够穿透嵌套页面执行指令。

### 2. 考试辅助系统

- **Vue 组件识别**：自动定位页面中的 Vue 实例及考试组件。
- 答案回填：
  - **选择题**：通过算法遍历选项组合或直接提取正确答案进行提交。
  - **填空题**：尝试从后端接口获取标准答案并自动填充。
- **解除限制**：移除部分页面禁止操作的限制，允许提前交卷。

### 3. 用户界面 (UI)

- **悬浮面板**：提供一个可拖拽的控制面板，方便用户随时调用功能。
- **视觉定制**：采用暗色调极简风格设计，支持自定义背景图片，界面硬朗、紧凑，不遮挡主要内容。

## 安装说明

1. 请确保您的浏览器已安装 **Tampermonkey** (油猴) 扩展程序。
2. 下载并安装本脚本。
3. 打开知行实验室相关课程页面，脚本将自动加载。

## 使用指南

脚本加载后，页面右上方会出现控制面板：

- **解决幻灯片**：点击后脚本将扫描当前课件的所有页面 ID，并批量发送已阅信号。完成后请刷新页面查看进度。
- **解决考试**：在考试页面点击此按钮，脚本将自动计算并填写答案。

## 配置说明 (自定义背景)

本脚本支持自定义面板背景图片。如需修改，请在脚本编辑器中找到代码顶部的 `CONFIG` 配置项：

```js
const CONFIG = {
    // 在此处替换您的图片链接
    bgImage: 'https://your-image-url.com/background.jpg',
    title: '知行实验室答题助手'
};
```

- **bgImage**: 填入图片的直链地址。若留空，则使用默认的深色背景。
- **title**: 可自定义面板顶部的标题文字。
